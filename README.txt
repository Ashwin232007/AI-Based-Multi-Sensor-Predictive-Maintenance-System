AI-Based Multi-Sensor Predictive Maintenance System
Overview
This project develops an intelligent predictive maintenance system for 3-phase induction motors using Machine Learning, Deep Learning, and Multi-Sensor Data Fusion. It detects bearing faults, estimates machine health, predicts Remaining Useful Life (RUL), and provides maintenance recommendations.
Features
    • Multi-Sensor Data Fusion
    • Bearing Fault Classification
    • Random Forest Model
    • CNN-BiLSTM with Attention
    • Hybrid Ensemble Learning
    • Machine Health Estimation
    • Remaining Useful Life (RUL) Prediction
    • Risk Score Calculation
    • Maintenance Recommendation Dashboard
Technologies Used
    • Python
    • TensorFlow
    • Keras
    • Scikit-learn
    • NumPy
    • Pandas
    • SciPy
    • Librosa
    • Matplotlib
Dataset
    • CWRU Bearing Dataset
    • Acoustic (.wav) Dataset
    • Electrical, Thermal, and Power Sensor CSV Data
Workflow
    1 Data Collection
    2 Signal Preprocessing
    3 Feature Extraction
    4 Multi-Sensor Feature Fusion
    5 Model Training (Random Forest & CNN-BiLSTM)
    6 Hybrid Ensemble Prediction
    7 Fault Diagnosis
    8 Machine Health & RUL Estimation
    9 Maintenance Recommendation
    10 Dashboard Visualization
Output
    • Bearing Fault Type
    • Fault Severity
    • Prediction Confidence
    • Machine Health (%)
    • Risk Score
    • Remaining Useful Life (RUL)
    • Maintenance Recommendation
    • System Alerts
